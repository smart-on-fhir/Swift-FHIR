    This method issues a POST, with the receiver serialized to JSON as the request’s body data, to the FHIR server. If the call succeeds (i.e. does not return an error), the receiver repopulates its properties with the server’s response (if the server is returning the resource, which some servers may not) and the _server property is assigned.
